# 🔥 Stealth Humanizer App

## Features
- ✅ Undetectable by AI detectors (GPTZero, Turnitin, Originality.ai)
- ✅ Multiple rewriting strategies per sentence
- ✅ Free & Open Source (HuggingFace models)
- ✅ Download results as .txt, .docx, or .pdf

## How to Run
1. Clone repo
2. `pip install -r requirements.txt`
3. `streamlit run app.py`

## How It Works
- Uses T5/BART for paraphrasing
- Back-translation for natural phrasing
- Burstiness & perplexity adjustment
- Contraction/filler injection
